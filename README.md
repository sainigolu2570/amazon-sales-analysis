## 📌 Project Overview
End-to-end data analysis of Amazon sales data to uncover
revenue trends, top-performing categories, geographic 
insights, and actionable business recommendations.

## 🎯 Business Questions Answered
- Which months and quarters generate the most revenue?
- Which product categories drive the most sales?
- Which states have the highest order volumes?
- What is the order cancellation rate?
- Which sales channels perform best?
